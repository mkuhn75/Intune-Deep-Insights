# Intune Deep Insights
The script can be run either in VS Code or directly in PowerShell 7.

## Software Requirements
- PowerShell 7

## PowerShell Module Requirements
- Install-Module Microsoft.Graph -Scope CurrentUser -Force
- Install-Module Microsoft.Graph.DeviceManagement -Scope CurrentUser -Force

## Usage
.\Intune-Deep-Insights-v2.ps1 -CustomerShortname TER

## Note
The authentication window may open in the background!
